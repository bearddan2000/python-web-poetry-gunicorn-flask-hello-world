# python-web-poetry-gunicorn-flask-hello-world

## Description
A POC for poetry dependencies resolver for secure production environments.

## Tech stack
- python
    - flask
    - gunicorn
- poetry

## Docker stack
- alpine:latest

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
